# Diffuse Qualitative-Oren-Nayar - Godot
This is an addon for Godot 4.4+ that adds the `Qualitative-Oren-Nayar` node to the visual shader system. This node outputs the `Diffuse Light` based on the `Qualitative-Oren-Nayar Diffuse Reflectance Model`.
